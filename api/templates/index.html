Financial Dashboard API

Welcome to the official API for the Financial Dashboard application. This backend service provides real-time financial data and news sentiment analysis.

API Endpoints

GET /api/financials/financial_data/<TICKER>/

Fetches comprehensive financial data for a specified company ticker from screener.in.


        


        

            

                GET
                /api/sentiment/analyze-sentiment/
            

            
Analyzes the sentiment of recent financial news for a given ticker or company name.
